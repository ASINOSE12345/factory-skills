# Architecture Decision Records

This directory records durable architecture decisions for `factory-skills`.

ADRs in this repository govern the toolkit that reads, audits, reasons over, and safely mutates Factory Memory. They should be written when a decision affects registry semantics, neuron scope, MCP behavior, governance hooks, write safety, privacy boundaries, or interoperability with FactoryOS.

## Records

| ADR | Status | Decision |
| --- | --- | --- |
| [ADR-0001](ADR-0001-factory-knowledge-architecture.md) | Proposed | Defines Factory as memory/intelligence platform, Factory Skills as cognitive toolkit, and FactoryOS as runtime consumer. |
